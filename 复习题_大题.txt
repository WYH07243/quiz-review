简答题

1. PCM时分复用系统：8路信号，每路最高频率4kHz，8位编码。求系统总码率和每时隙宽度。
我的答案:采样频率fs=2x4=8kHz，帧周期Ts=1/8000=125μs，每帧8时隙，每时隙8bit，总码率Rb=8x8000x8=512kbit/s，每时隙宽度=125/8=15.625μs
正确答案:Rb=512kbit/s，时隙宽15.625μs

2. 单路PCM编码：模拟信号最高频率3.4kHz，A律13折线PCM编码，求编码速率。
我的答案:采样频率fs=2x3.4=6.8kHz，取标准8kHz，n=8bit，Rb=8000x8=64kbit/s
正确答案:64kbit/s

3. A律13折线编码：输入+635，求8位PCM编码。
我的答案:极性码1，段落码110(512-1024段)，段内码：635-512=123，123/32=3余27，段内码0011，完整编码：11100011
正确答案:11100011

4. 升余弦滚降系统：码元速率2400Baud，滚降系数0.4，求系统带宽和频带利用率。
我的答案:带宽B=(1+0.4)x2400/2=1680Hz，频带利用率η=2400/1680=1.43Baud/Hz
正确答案:1680Hz，1.43Baud/Hz

5. AMI/HDB3编码：信息码10000110100001，求AMI码和HDB3码。
我的答案:AMI:+1 0 0 0 0 -1 +1 0 -1 0 0 0 0 +1；HDB3：第1组4连0替换为000V，第2组替换为B00V
正确答案:AMI:+1,0,0,0,0,-1,+1,0,-1,0,0,0,0,+1

6. 卡森公式：调频信号Δf=75kHz，fm=15kHz，求FM信号带宽。
我的答案:BFM=2(Δf+fm)=2(75+15)=180kHz，或mf=5，BFM=2(5+1)x15=180kHz
正确答案:180kHz

7. 香农公式应用：解释为什么FM可以在低信噪比下工作。
我的答案:香农公式C=Blog2(1+S/N)，通过增加带宽B可以降低对S/N的要求，FM正是用带宽换信噪比，将信号扩展到宽带，使发送功率谱密度降低，在接收端通过解调增益G=3mf^2(mf+1)提升信噪比
正确答案:FM通过宽带传输换取信噪比改善

8. 平稳随机过程：自相关函数RX(τ)=16+1/(1+τ²)，求均值和方差。
我的答案:E[X]=±√RX(∞)=±√16=±4；均方值RX(0)=16+1=17；方差=17-16=1
正确答案:均值±4，方差1

9. 白噪声通过带限系统：n₀/2=10⁻¹⁰W/Hz，B=10kHz，求输出噪声功率。
我的答案:Nout=n₀B=2x10⁻¹⁰x10⁴=2x10⁻⁶W
正确答案:2x10⁻⁶W

10. 信源熵：符号集A,B,C,D,E概率1/2,1/4,1/8,1/16,1/16，求信源熵和信息速率(1000符号/s)。
我的答案:H=-(0.5log0.5+0.25log0.25+0.125log0.125+0.0625log0.0625x2)=1.875bit/符号；Rb=1.875x1000=1875bit/s
正确答案:1.875bit/符号，1875bit/s

11. 2PSK相位模糊：为什么实际用2DPSK而不用2PSK？
我的答案:2PSK相干解调需要恢复同频同相载波，若本地载波与发送载波反相，解调结果0↔1全部翻转（倒π现象）。2DPSK用前后码元相对相位变化传信息，解调只需比较相邻码元相位差，与绝对相位无关，避免相位模糊问题
正确答案:2DPSK避免相位模糊

12. 眼图作用：简述眼图的4个作用。
我的答案:①衡量码间串扰大小（眼开越大串扰越小）；②反映噪声容限（眼开高度）；③确定最佳抽样时刻（眼开最宽处）；④判断定时误差灵敏度（斜边斜率）
正确答案:4个作用：码间串扰、噪声容限、抽样时刻、定时误差

13. 制度增益G：比较DSB、SSB、FM的制度增益。
我的答案:DSB:G=2；SSB:G=1；FM(大信噪比):G=3mf²(mf+1)，FM的G远大于DSB/SSB，但占用带宽大
正确答案:DSB:G=2，SSB:G=1，FM:G=3mf²(mf+1)

14. 奈奎斯特第一准则：解释其含义。
我的答案:无码间串扰条件：系统传输函数H(ω)以2π/Ts为周期平移后叠加在(-π/Ts,π/Ts)内为常数。系统最大传输速率RB=2Baud/Hz，当B=RB/2时达到极限
正确答案:奈奎斯特第一准则描述无码间串扰条件

15. 均匀量化vs非均匀量化：比较两者特点。
我的答案:均匀量化：量化台阶固定，小信号信噪比低，大信号信噪比高；非均匀量化：小信号台阶小，大信号台阶大，在宽动态范围内获得接近一致的信噪比。实现：发送端压缩再均匀量化（A律/μ律），接收端扩张恢复
正确答案:非均匀量化提升小信号信噪比
