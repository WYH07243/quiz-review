简答题

1. 已知二进制线性分组码的码字集合为{00000, 01101, 10110, 11011}，求该码的最小汉明距离，并说明它能检测几位错误？纠正几位错误？
解：计算两两汉明距离：
d(00000,01101)=3
d(00000,10110)=3
d(00000,11011)=4
d(01101,10110)=? 01101⊕10110=11011，重量3
d(01101,11011)=10110? 01101⊕11011=10110，重量3
d(10110,11011)=01101，重量3
所以最小距离d_min=3。
检测错误能力：e = d_min-1 = 2位。
纠正错误能力：t = floor((d_min-1)/2)=1位。
答：d_min=3，可检2错，纠1错。
2. 什么是汉明距离？最小汉明距离与纠检错能力的关系是什么？
答：汉明距离是指两个等长码字之间不同码元的个数。码的最小汉明距离d_min是码字集合中任意两个不同码字间的最小距离。关系：
检测e个随机错误：d_min ≥ e+1
纠正t个随机错误：d_min ≥ 2t+1
同时检测e个、纠正t个错误（e≥t）：d_min ≥ e+t+1
3. 比较均匀量化与非均匀量化的性能，说明非均匀量化的实现方法。
答：均匀量化在信号动态范围内量化台阶固定，缺点是小信号量化信噪比低，大信号信噪比高但不必要；非均匀量化使小信号台阶小、大信号台阶大，从而在整个动态范围内获得接近一致的量化信噪比。实现方法：在发送端将信号进行压缩（如对数压缩），再进行均匀量化，等效为非均匀量化；接收端做扩张，恢复线性关系。工程上常用A律13折线或μ律15折线近似对数特性。
4. 设PCM时分复用系统有8路，每路信号最高频率4kHz，采用8位编码，试求该系统的总数码率和每帧的时隙宽度。
解：抽样频率f_s = 8kHz，帧周期T_s = 1/f_s = 125μs。每帧有8个时隙（每路一个），每个时隙传8位码。总数码率 R_b = 8路 × f_s × 8 = 8×8000×8 = 512 kbit/s。每帧位数 = 8×8=64位，帧长125μs。每个时隙宽度 = 125μs/8 = 15.625μs。
答：总数码率512 kbit/s，时隙宽度15.625μs。
5. 设单路模拟信号最高频率为3.4kHz，采用A律13折线PCM编码，求该PCM系统的数码率。
解：由抽样定理，f_s ≥ 2×3.4 = 6.8 kHz，通常取f_s = 8 kHz。A律13折线编码n=8位。数码率 R_b = f_s × n = 8×8 = 64 kbit/s。
答：64 kbit/s。

7. 简述低通信号的抽样定理。
答：对于频带受限的模拟信号m(t)，若其最高频率为f_H，当抽样频率f_s ≥ 2f_H时，可以从抽样信号中无失真地恢复原信号。最低抽样频率2f_H称为奈奎斯特频率。若f_s < 2f_H，则产生频谱混叠，无法恢复原信号。
8. 什么是数字信号的最佳接收？它与传统接收有何区别？
答：数字信号的最佳接收是以最小差错概率为准则，在统计检测理论基础上设计接收机，充分利用信号和噪声的统计特性，使得判决的误码率最低。传统接收（如滤波、抽样判决）通常以输出信噪比最大为准则，而最佳接收直接从最小误码率出发，在AWGN信道下匹配滤波器/相关接收机与传统接收等价，但在非白噪声或非确知信号条件下，最佳接收更优越。
9. 为什么实际通信中常采用2DPSK而不直接采用2PSK？
答：2PSK相干解调需要接收端恢复同频同相的相干载波，但载波同步电路存在相位模糊问题（无法区分0和π相位），导致解调后码元全部倒置。2DPSK通过将绝对码变换为相对码，用前后码元载波相位差传输信息，解调时只需比较相邻码元相位差，与绝对相位无关，从而避免了相位模糊。虽然2DPSK的误码性能比2PSK略有下降，但工程实现更可靠。
10. 为什么2PSK相干解调会出现“相位模糊”现象？如何解决？
答：
原因：2PSK信号的载波相位只有0和π两种，代表“1”和“0”。接收端的相干载波由载波同步电路提取，但同步电路无法区分载波的绝对相位（0或π），可能输出与发送载波同相或反相两种结果，导致解调后所有码元极性取反，即“0”判为“1”，“1”判为“0”，称为反向工作或相位模糊。
解决方法：采用差分编码，将绝对码转换为相对码，然后进行2PSK调制（实际是2DPSK），接收端通过比较相邻码元载波相位差来判决，与绝对相位无关，从而消除模糊。
11. 设基带传输系统总频率特性为升余弦滚降特性，滚降系数α=0.4，若码元传输速率为2400 Baud，试求：
（1）系统带宽；
（2）频带利用率；
解：已知 R_B = 2400 Baud，α=0.4。
（1）升余弦滚降系统带宽 B = (1+α)R_B/2 = (1+0.4)×2400/2 = 1.4×1200 = 1680 Hz。
（2）频带利用率 η_B = R_B / B = 2400 / 1680 ≈ 1.4286 Baud/Hz（或表示为 2/(1+α) = 2/1.4 ≈ 1.4286）。
12. 设二进制序列为 1 0 0 0 0 1 1 0 100001，试编出AMI码和HDB3码并画出归零波形。（设第一个1和V都为正极性）
答：AMI码：+1, 0, 0, 0, 0, -1, +1, 0, -1
HDB3码：检测连续4个“0”，第4个“0”替换为破坏脉冲，破坏脉冲极性与前一个非零码元相同（或相反，依规则）。详细波形需根据规则画出。
（注：波形可示意：对序列 1 0 0 0 0 1 1 0 1，前4个“0”替换为“000V”或“B00V”，依据前一个非零码元极性。例如第一个“1”为+V，则前4个零替换为“000+V”，后续“1”为-1，再下一个“1”为+1，接着“0”为0，最后“1”为-1。）
13. 简述眼图的概念及其作用。
答：眼图的作用：
直观评估基带传输系统的性能；
估算码间串扰大小（眼睛张开度越小，串扰越大）；
估计噪声容限（眼睛垂直开口高度）；
确定最佳抽样时刻（眼睛张开最大时刻）；
确定定时误差敏感度（眼图斜边斜率）。
14. 什么是奈奎斯特第一准则？其物理含义是什么？
答：奈奎斯特第一准则（无码间串扰准则）指出：若基带系统的总频率特性H(ω)满足
∑n=?∞∞H(ω+2nπTs)=Ts,∣ω∣≤πTsn=?∞∑∞?H(ω+Ts?2nπ?)=Ts?,∣ω∣≤Ts?π?
则系统能以最高为 2/T_s 的码元速率传输且无码间串扰。其中T_s为码元间隔。
物理含义：将H(ω)以2π/T_s为间隔左右平移叠加，若在(-π/T_s, π/T_s)范围内叠加结果为常数，则无码间串扰。这意味着系统总带宽B至少为R_B/2，且当B = R_B/2时达到极限。
15. 调频信号带宽的卡森公式是什么？其中各参数的含义是什么？
答：Δf为最大频偏，fm为调制信号的最高频率，mf= Δf /fm为调频指数，表示调频的强弱程度。
16. 设某信道具有均匀的双边噪声功率谱密度P_n(f) = n_0/2 = 0.5×10?3 W/Hz，基带信号带宽为5kHz，采用SSB调制方式传输，载频为100kHz，已调信号功率为10kW。若接收端的输入信号先经过一个理想带通滤波器，求相干解调时解调器输出端的信噪比。

17. 如何比较两个模拟通信系统的抗噪声性能？
答：在已调信号平均功率相同、信道噪声功率谱密度相同的条件下，输出信噪比直接反映了解调器的抗噪声性能。为便于比较同类调制系统采用不同解调器时的性能，还常用调制制度增益?G =（输出信噪比）/（输入信噪比）来衡量。制度增益越大，说明解调器的抗噪声性能越好。
18. 什么是频率调制？什么是相位调制？两者关系如何？

答：频率调制（FM）：瞬时频率偏移随调制信号成比例变化的调制方式。

相位调制（PM）：瞬时相位偏移随调制信号成比例变化的调制方式。

两者关系：调频和调相统称为角度调制。调频信号相当于调制信号先微分后再调相，调相信号相当于调制信号先积分后再调频，两者可互相转化。
19. 从香农公式出发，解释为什么扩频通信可以在低信噪比下工作。
答：香农公式 C = B log?(1 + S/N) 表明，在固定信道容量C下，可以通过增加带宽B来降低对信噪比S/N的要求。扩频通信将信号扩展到很宽的频带，使得在发送功率一定的情况下，信号功率谱密度很低，接收端的信噪比可以远小于1（负dB），但总容量仍能满足通信速率要求。这就是扩频通信抗干扰、保密、低截获的基础。

21*. 为什么在实际数字通信系统中，信道容量通常远小于香农极限？
答：香农极限是在理想条件下（高斯白噪声、无限长随机码、最佳译码）的容量。实际系统距离该极限有差距的原因包括：
调制方式和编码方案非理想，如QAM、PSK等有性能损失；
实际信道不仅存在高斯噪声，还有脉冲干扰、非线性失真、衰落等；
接收机同步、量化、实现损失；
有限码长和译码复杂度限制；
为满足一定误码率，需要留出信噪比余量。
近年来Turbo码、LDPC码已能极接近香农极限（相差不足0.1dB）。
22. 什么是各态历经性？工程上为什么常常假设随机过程具有各态历经性？
答：各态历经性是指平稳随机过程的任一样本函数的时间平均（如时间均值、时间自相关）等于该过程的集合平均（统计均值、统计自相关）。工程上常常假设各态历经性，因为实际中只能获得一个样本函数（一次实验记录），若过程是各态历经的，就可以用一个样本的时间平均来代替大量样本的统计平均，大大简化测量和分析。
